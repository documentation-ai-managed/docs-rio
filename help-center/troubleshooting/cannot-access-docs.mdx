---
title: Cannot Access Documentation
description: Quick fix if you can't view documentation pages.
---

## Symptoms

- Docs page shows "Access Denied"
- Unable to open certain documents

## Causes

- Insufficient permissions
- You're not signed in

## Solution

<Steps>
  <Step title="Check login status">
    ````
    Make sure you are logged into Rio. If not, sign in.
    ````
  </Step>
  <Step title="Request access">
    ````
    Contact your project admin for permissions if needed.
    ````
  </Step>
</Steps>
