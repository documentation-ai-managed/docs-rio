## Prerequisites

Before starting, ensure you have:

<Callout kind="info">
- A modern web browser like Chrome, Firefox, or Safari
- An email address for account verification
- Optional: GitHub account for version control integration
</Callout>

<Callout kind="tip">
Rio works entirely in your browser. No downloads required.
</Callout>

## Create Your Rio Account

Follow these steps to sign up and verify your account.

<Steps>
  <Step title="Visit Rio" icon="globe">
    Navigate to `https://app.rio.com` and click **Sign Up**.
  </Step>
  <Step title="Enter Details" icon="user">
    Provide your email, name, and a strong password. Accept the terms.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox for the verification link from Rio and click it.
  </Step>
  <Step title="Complete Onboarding" icon="check-circle">
    Choose your workspace name and select your preferred theme.
  </Step>
</Steps>

## Set Up Your First Documentation Space

Once logged in, create a dedicated space for your project docs.

<Steps>
  <Step title="Create Space" icon="plus">
    Click **New Space** in the dashboard. Enter a name like `My Project Docs`.
  </Step>
  <Step title="Configure Settings" icon="settings">
    Set visibility to **Private** or **Public**, and enable search indexing.
  </Step>
  <Step title="Add Description" icon="file-text">
    Write a brief overview: "Documentation for My Project, including API guides and changelogs."
  </Step>
</Steps>

## Add and Organize Initial Pages

Rio supports both visual editing and MDX for advanced formatting. Use the method that suits you.

<Tabs>
  <Tab title="Visual Editor" icon="edit-3">
    <Steps>
      <Step title="New Page" icon="plus">
        In your space, click **New Page** and select **Blank Page**.
      </Step>
      <Step title="Add Content" icon="type">
        Use the toolbar to add headings, lists, and embeds. Drag to reorganize.
      </Step>
      <Step title="Publish" icon="upload">
        Click **Publish** to make it live. Share the page URL.
      </Step>
    </Steps>
  </Tab>
  <Tab title="MDX Editor" icon="code">
    Create rich pages with components.

    <CodeGroup tabs="JavaScript,Python">
    ````javascript
    // Example API call to fetch space data
    const response = await fetch('https://api.example.com/spaces/my-project', {
      headers: { Authorization: `Bearer ${YOUR_API_KEY}` }
    });
    const space = await response.json();
    console.log(space.pages);
    ````
    ````python
    import requests

    response = requests.get(
        'https://api.example.com/spaces/my-project',
        headers={'Authorization': f'Bearer {YOUR_API_KEY}'}
    )
    space = response.json()
    print(space['pages'])
    ````
    </CodeGroup>
  </Tab>
</Tabs>

## Basic Navigation and Editing Tools

Master these essentials for efficient workflows.

| Tool | Shortcut | Description |
|------|----------|-------------|
| Search | `<kbd>Ctrl</kbd>+<kbd>K</kbd>` | Find pages and content instantly |
| New Page | `<kbd>Ctrl</kbd>+<kbd>N</kbd>` | Quick page creation |
| Toggle Preview | `<kbd>Ctrl</kbd>+<kbd>P</kbd>` | Switch between edit and preview modes |
| Outline | Sidebar icon | Jump to headings |

<Expandable title="Advanced Editing Tips" default-open="false">
- Use `@mentions` for team collaboration
- Embed Figma or YouTube with `/embed`
- Version history via **Revisions** panel
</Expandable>
